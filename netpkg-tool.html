Package Your .NET Apps On Linux

Downloading is easy!

Releases can be downloaded directly from Github. Use the version tag to specify which one.

wget https://github.com/phil-harmoniq/netpkg-tool/releases/download/0.3.8/netpkg-tool

chmod a+x ./netpkg-tool

./netpkg-tool --help

If you're feeling adventurous,

you can easily build netpkg-tool from source.

git clone https://github.com/phil-harmoniq/netpkg-tool

cd netpkg-tool && ./build.sh .

./netpkg-tool --help

Also available as a Docker image!

docker run --rm philharmoniq/netpkg-tool --help

<3 100% Open Source <3

Details

Using netpkg-tool will restore and compile your project based on settings in your *.csproj file. By default, netpkg-tool will use Framework Dependent Deployment to compile your project. To use Self-Contained Deployment, use the --scd flag. You aplications will then be packaged up into an AppImage. (More package formats to come)

Get more info from the Github repo!

If you'd like to dig deeper into the development of netpkg-tool, all of our source is available on Github. You can also see all of our build logs on Travis CI.
