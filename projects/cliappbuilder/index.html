A simple, yet flexible CLI creation library for .NET Core

CliAppBuilder

Streamline the creation of CLI applications using helpful builder classes. CliAppBuilder is designed to be simple, yet flexible.

Installing

Install using the .NET CLI or your IDEâ€™s package manager:

dotnet add package CliAppBuilder

Usage

Begin by creating a new CliBuilder() and add your flags, values, and actions by using the included helper methods. There are additional methods for updating the meta-data of your application such as name, version, default actions, and more. Try exploring the API on FuGet!

Provide an anonymous function to a new ActionBuilder() to add functionality to your CLI application. The arguments provided to your function are:

  1. The ActionArg that was called to provide the context for your application.
  2. The additional arguments given to that action after parsing any detected special arguments.

The provided ActionArg provides many properties storing meta-data for that action as well as a reference to the base CliApp for additional utility methods and properties.

For more detailed instructions please visit the CliAppBuilder wiki.

Example


static void Main(string[] args)
{
    var app = new CliBuilder()
        .SetName("CliAppBuilder Example")
        .SetWidth(72)
        .AddActionArg(new ActionBuilder("Print to the terminal", "print", PrintToTerminal)
            .AddFlagArg("Print with the color red", "--red", "-r")
            .AddFlagArg("Print with the color blue", "--blue", "-b")
            .AddFlagArg("Print with the color green", "--green", "-g")
            .AddValueArg("Log what you said in the given text file", "--log", "-l")
        .AddActionArg(new ActionBuilder("List files in current directory", "list", ListFiles)
            .AddFlagArg("Show hidden files", "--hidden", "-h"))
        .DontFailOnInvalidArg()
        .Start(args);
}
